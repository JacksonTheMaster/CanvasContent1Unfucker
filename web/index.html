How to Use

  1. Enter your CanvasContent1 Soup in the first box.
  2. Click "UnSoup" to decode and format the Json.
  3. Edit the Json in the Json Output box as needed, or copy it to the raw Output box for real time Json Validation.
  4. OR Copy the output to VS Code. Either use the Json Output box (easy, should work nicely), or use the Raw Output box and format it using Prettier (Ctrl+A, Ctrl+K, Ctrl+F).
  5. Edit your JSON as needed.
  6. Copy the formatted, edited JSON back into the second box. Either from VsCode, or from the Json Output box depending on where you did your editing.
  7. Click "reSoup" to minify and convert it back to its original format.
  8. Only the seccond box is reSouped, the Json Output box is left unread.
  9. Use the minified JSON in your API call.
  10. Enjoy your soup.

Frustrated with wading through Microsoft's complex and messy nested JSON structures in SharePoint APIs, I created this tool to simplify the process. The SharePoint CanvasContent1 Unfucker was born out of the need to make these JSON soups more readable and easier to work with. Enjoy your soup, served just the way you like it!

You can manually decode your soup by replacing all occurrences of \\\ with a temporary marker, then replacing all occurrences of \" with ", and finally replacing the temporary marker back to \. To reverse the process, simply follow these steps in reverse order.
